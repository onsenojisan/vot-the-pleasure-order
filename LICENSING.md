## Why `LICENSE` holds the full legal text

`LICENSE` previously named CC-BY-4.0 and linked to the legal code rather than reproducing it.
That is a valid grant for a human reader, but GitHub and automated license scanners identify a
license by matching the file against the known legal text, so the repository was reported as
`NOASSERTION` — "no license detected". `LICENSE` now carries the full text so the license is
machine-readable, and this file carries the attribution and the per-record notes.
