## Current Scope

This repository is intended to hold public-facing restart materials only.

Current public scope:

- stable restart status;
- public claim boundaries;
- public-facing definition and readiness summaries;
- clear separation between current restart material and older Omega-facing materials.

This repository does not contain empirical result tables, metrics, notebooks, CSV templates, datasets, or validation outputs.

## Status Of Earlier Omega Materials

Earlier Omega empirical and reproducibility repositories remain available as historical, methodological, or exploratory records unless explicitly reauthorized.

They should not be read as current validation of Omega, and they are not the current project center.

## Validation Status

Omega validation remains blocked.

Any future empirical claim would require fixed definitions, independent event labels, non-circularity checks, sufficient sample size, and a validation design before evaluation.

No new empirical claim is made by this repository.
